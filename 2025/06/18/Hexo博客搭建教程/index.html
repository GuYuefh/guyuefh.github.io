六、部署到 Vercel

官网地址

使用 GitHub 注册并登录，选择 Add New > Project

找到之前创建的库，点后面的 Import，直接点 Deploy 等待部署完成

之后到 Project 页面 > Domains 添加自己的域名，在域名管理页解析即可通过域名访问了！

建议购买自己的域名，推荐阿里云。
