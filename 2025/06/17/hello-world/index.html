Hexo博客搭建教程
一、环境准备 Node.js 安装Node.js 是 Hexo 的运行环境，必须先安装。 I、前往官网下载II、默认选项安装III、打开命令提示符(CMD)验证安装: 12$ node -v# 检查 Node.js 版本 返回版本信息则代表安装成功！ 然后进行npm换源，运行 1$ npm config set registry https://registry.npmmirror.com 无报错则安装完成！ Git 安装 I、下载 GitII、安装时选择默认选项III、验证安装: 1$ git --version 安装 Hexo Hexo官网 在所有应用中打开安装好的 Git Bash 运行 1$ npm install -g hexo-cli 进行安装，之后运行 1$ hexo -v 验证是否安装成功 GitHub 部署注册后创建仓库名格式必须为：<用户名>.github.io 最好创建成公开库 Git Bash 运行 12$ git config --global user.name <你的用户名>$ git config...
